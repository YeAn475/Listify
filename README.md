## 📌 프로젝트 배경
음악 스트리밍 서비스 이용이 증가하면서  
사용자들은 단순한 음악 감상을 넘어 **플레이리스트를 공유하고 추천받는 경험**을 원하고 있습니다.  

기존 서비스들은 개인화 추천은 강력하지만  
**플레이리스트 공유와 큐레이션 기능은 제한적**이라는 한계가 있었습니다.

본 프로젝트는 이러한 문제를 해결하고자  
**사용자 취향 기반 플레이리스트 추천 및 공유 플랫폼**을 기획하게 되었습니다.

---

## 📝 프로젝트 소개
Listify는 사용자 취향을 기반으로 음악을 추천하고,  
플레이리스트를 생성·공유할 수 있는 **웹 기반 음악 큐레이션 서비스**입니다.

- 개인 맞춤형 플레이리스트 추천
- 플레이리스트 공유 및 복사 기능
- 직관적인 UI/UX 제공

---

## 🎯 프로젝트 목표
- 사용자 취향을 반영한 **개인화 추천 서비스 구현**
- 직관적이고 사용하기 쉬운 **UI/UX 설계**
- 프론트엔드 / 백엔드 **분리 아키텍처 경험**
- 클라우드 환경에서의 **실제 배포 경험**

---

## ⭐ 주요 기능
- 🔍 음악 검색 기능
- ❤️ 플레이리스트 생성 및 저장
- 📋 플레이리스트 공유 및 복사
- 🎧 사용자 취향 기반 추천
- 🔐 로그인 / 인증 처리
- 📱 반응형 UI 지원
